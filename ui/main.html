Help Layers

  • Make All - Makes all the help layers.
  • Memo - Makes layer for writing notes.
  • Perspective Lines - Lines to check perspective.
  • Solarisation - For finding dust spots and composition errors.
  • Skin Checker - For finding skin imperfections.
  • Lightness - For finding lumonosity differences eg. in dodging & burning.
  • Saturation Map - For finding saturation differences.
  • Match Tones - For finding tone differences.

Layers Saver

This makes layers animation from bottom to top which shows how editing progresses. You can mark the main layer with # and it will be saved first. If main layer is a smart object it is copied and opened so you can reset eg. lightroom settings.
